
# EEstring

End - to - End Encrypted Message Exchange Server and client by PGP key

## Compile

### Make Test Certificate
```bash
    make init 
```

### Make server, cli, all
```bash
    make server
    make cli
    make
```

### Clean all
```bash
    make fclean
```
